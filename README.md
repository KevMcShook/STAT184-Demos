# STAT184-Demos
Repo for in-class demos in STAT184. Demonstrating basic functionality of GitHub and as part of a reproducible workflow for stats and data science.

## Data

There are no data associated with this repo.

## Current Plan

Within this repo, we will be practicing the following:

1. Making repos
2. Making commits with informative subjects and messages
3. Using branches
4. Setting up initial README
5. Use of the issues feature

## Organization

This repo will use a simple organization structure; all files available in the top-level folder without any subfolders
